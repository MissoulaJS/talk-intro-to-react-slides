The render method

  • Pure Function
  • Don't change props or state
  • Return hierarchy of elements

The render method


  var HelloMessage = React.createClass({
    render: function() {
      var name = 'Mr. ' + this.props.name;
      return (
        <div>
          Hello {name}!
        </div>
      );

    }
  });

props and state

Two types of model data

props and state

  • Set props from the outside
  • Set state from the inside
  • Try to avoid state

props and state


  var HelloMessage = React.createClass({
    componentWillMount: function() {
      this.setState({ time: new Date().toLocaleTimeString() });
      self = this;
      setInterval(function() {
        self.setState({ time: new Date().toLocaleTimeString() });
      }, 1000);
    },
    render: function() {
      return (
        <div>
          Hello! The time is {this.state.time}.
        </div>
      );
    }
  });

Global State

Set data-state="something" on a slide and "something" will be added as a class to the document element when the slide is open. This lets you apply broader style changes, like switching the page background.

State Events

Additionally custom events can be triggered on a per slide basis by binding to the data-state name.

  console.log( '"customevent" has fired' );
